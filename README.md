# Image Compressor Tool

A modern and free online Image Compressor that allows users to reduce image file sizes directly in the browser without uploading files to a server.

## Live Demo

https://your-project.vercel.app

## Project Overview

Large image files can slow down websites, increase storage usage, and make file uploads difficult. This Image Compressor helps users optimize images quickly while maintaining acceptable visual quality.

The tool performs all compression operations directly in the browser, ensuring faster performance and better privacy.

## Features

* Upload images using file picker or drag-and-drop
* Support for JPG, JPEG, PNG, and WEBP formats
* Real-time image preview
* Adjustable compression quality
* Display original file size and compressed file size
* Calculate percentage of space saved
* Download compressed images instantly
* Responsive design for desktop and mobile devices
* Dark and Light mode support
* Batch image compression
* Compression history using localStorage

## Installation

Clone the repository:

```bash
git clone https://github.com/Preetika26/image-compressor-tool.git
```

Navigate to the project directory:

```bash
cd image-compressor-tool
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deployment

This project is deployed using Vercel.

To deploy:

1. Push the project to GitHub.
2. Connect the repository to Vercel.
3. Import the project.
4. Deploy with default Vite settings.

## Project Motivation

As a web development student, I frequently needed to upload images for projects, assignments, and websites. Many existing image compression tools contained ads, restrictions, or required subscriptions. To solve this problem, I built a free and easy-to-use Image Compressor that works directly in the browser and provides instant results.
